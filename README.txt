##### README for this folder #####
This folder contains all the source files, along testing input files that are my 
responses to the code challenges in the textbook "Bioinformatics Algorithms: An
Active Learning Approach". All these code challenges can also be viewed from the 
textbook track of the Rosalind webpage: 
http://rosalind.info/problems/list-view/?location=bioinformatics-textbook-track.
